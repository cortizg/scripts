This page lists characters in the following Unicode block and provides information about them.

  • Tai Tham

This is not authoritative, peer-reviewed information – these are just notes I have gathered and copied from various places.

For a summary of the script and its use in writing systems, see the page Tai Tham script summary. For similar information related to other scripts, see this list.

We have usage data for no languages that use the Tai Tham script.

In order to allow the fonts to work on all platforms, rather than use '-' as a generic base for vowels and diacritics, this document uses the character WA, which is a simple circle. The transliterations are home-grown and language-specific. If you see two transliterations after the type description, the first is for Northern Thai, the second for Khün.

Tone marks

If there is a vowel over a consonant or consonant stack, the tone mark follows the vowel in storage, and is displayed above or alongside the vowel.

Otherwise, the tone is input after the consonant, ie. before a vowel sign that is displayed to the right or below, and appears over the consonant. e

The fonts i have expect the tone to be typed after a lefted vowel if there is one; after a vowel above, if there is one; before a vowel to the right; and doesn't seem to matter wrt low vowel. See this test. Noto agrees except for lefted vowels.

Written over a consonant (normally in final position) when that consonant is not to be pronounced. Frequently used in loans from languages with consonant clusters in the coda such as Pali, eg. ᩈᩫ᩠ᨾᨷᩪᩁ᩠᩼ᨱ sɔ̈˖m̱b̯ūr˖˚ṇ̱ som¹buːn² perfect or English ᨼᩥ᩠ᩃ᩼ᨾ f̱i˖ḻ˚m̱ fim² film. o p149

U+1A7B TAI THAM SIGN MAI SAM

See in UniView

Northern Thai

Used at the end of a word to indicate that it should be repeated, eg. ᨲ᩵ᩣ᩠ᨦ t¹ā˖ŋ̱ taːŋ to be different, vs. ᨲ᩵ᩣ᩠ᨦ᩻ t¹ā˖ŋ̱ʻ taːŋ taːŋ different in my view. e

Also used to indicate 'double-acting' consonants, stored where the consonant would be stored if there were a separate consonant used, eg. compare ᨡᩮᩢ᩶ᩣᨡᩬᨦ xeá²āxɔ̱ŋ̱ xo²xɔŋ belongings with an alternative spelling using this diacritic, ᨡᩮᩢ᩶᩻ᩬᩣ᩠ᨦ xeá²ʻɔ̱ā˖ŋ̱. e

A third use is somewhat different. A subjoined letter may be medial, final, or the start of a new syllable. Mai sam can be used to indicate that a consonant begins a new syllable, in which case it is stored following the subjoined form and indicates that it is the consonant that starts the new syllable, eg. compare ᨳᩫ᩠ᨶᩁ tʰɔ̈˖ṉr tʰonra with ᨳᩫ᩠ᨶ᩻ᩁ tʰɔ̈˖ṉʻr tʰanon path (final r is pronounced as n). e

General sources: Everson, Owen
