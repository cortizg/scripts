Usage & history

For information about the script in general, see the Arabic overview.

Kashmiri is written in the devanagari script by Hindus. Muslims use the arabic script.

See the comparison table

Distinctive features

The Arabic script is an abjad. This means that in normal use the script represents only consonant and long vowel sounds. The Arabic alphabet has been adapted to cope with the many more vowels sounds in languages such as Kashmiri. See the table to the right for a brief overview of features, taken from the Script Comparison Table.

Kashmiri is principally written using the nasta'liq style of Arabic writing. Glyphs are more drawn out, and the baseline tends to be sloping from word to word.

Vowels

Phoneme table

The allocation of characters to vowel sounds is somewhat complicated. The complexity arises from the number of vowels in Kashmiri compared to the Arabic language, and the need to represent them all, but also because different sequences are needed for different positional forms, and also there are often a number of alternative character sequences possible. For more detailed information, see the following section.

vowels

Vowels in initial or isolated position are written with a preceding ا [U+0627 ARABIC LETTER ALEF].

The following descriptions should be read with the vowel phoneme chart in the previous section.

i is written with ِ [U+0650 ARABIC KASRA].

ī uses ی + ٖ [U+06CC ARABIC LETTER FARSI YEH + U+0656 ARABIC SUBSCRIPT ALEF] for initial and medial positions, but just ی [U+06CC ARABIC LETTER FARSI YEH] for final and isolate.

ɨ uses ٕ [U+0655 ARABIC HAMZA BELOW]. The initial and isolate forms can be rendered with the precomposed character إ [U+0625 ARABIC LETTER ALEF WITH HAMZA BELOW], which is canonically equivalent to ا + ٕ [U+0627 ARABIC LETTER ALEF + U+0655 ARABIC HAMZA BELOW] .

ɨː uses ٟ [U+065F ARABIC WAVY HAMZA BELOW]. There is a precomposed character for the initial and isolate combinations with alef, ٳ [U+0673 ARABIC LETTER ALEF WITH WAVY HAMZA BELOW], that neither composes nor decomposes in normalisation, but it is strongly deprecated by the Unicode Standard.

u uses ُ [U+064F ARABIC DAMMA].

uː uses و + ٗ [U+0648 ARABIC LETTER WAW + U+0657 ARABIC INVERTED DAMMA].

e is unclear. Wikipedia uses ێ [U+06CE ARABIC LETTER YEH WITH SMALL V] throughout, but Daniels and LOC show a YEH with what looks like a combining breve above for the initial and medial forms, and ے [U+06D2 ARABIC LETTER YEH BARREE] with the same diacritic above for final and isolate.

eː uses ی [U+06CC ARABIC LETTER FARSI YEH] for initial and medial forms, and ے [U+06D2 ARABIC LETTER YEH BARREE] for final and isolate.

ə uses ٔ [U+0654 ARABIC HAMZA ABOVE], however there are several precomposed characters available for use with hamza above. These include:

  • أ [U+0623 ARABIC LETTER ALEF WITH HAMZA ABOVE]
  • ؤ [U+0624 ARABIC LETTER WAW WITH HAMZA ABOVE]
  • ئ [U+0626 ARABIC LETTER YEH WITH HAMZA ABOVE]
  • ۂ [U+06C2 ARABIC LETTER HEH GOAL WITH HAMZA ABOVE]
  • ځ [U+0681 ARABIC LETTER HAH WITH HAMZA ABOVE]
  • ݬ [U+076C ARABIC LETTER REH WITH HAMZA ABOVE]
  • ࢡ [U+08A1 ARABIC LETTER BEH WITH HAMZA ABOVE]

All of the above except HAH, REH, and BEH decompose and recompose during normalisation.

əː uses ٲ [U+0672 ARABIC LETTER ALEF WITH WAVY HAMZA ABOVE].

o uses ۆ [U+06C6 ARABIC LETTER OE].

oː uses و [U+0648 ARABIC LETTER WAW].

a uses َ [U+064E ARABIC FATHA].

aː uses آ [U+0622 ARABIC LETTER ALEF WITH MADDA ABOVE] in initial and isolate forms, and just ا [U+0627 ARABIC LETTER ALEF] elsewhere. The former may be decomposed.

ɔ uses ۄ [U+06C4 ARABIC LETTER WAW WITH RING].

ɔː is used only in isolated and final forms, and uses ۄ + آ [U+06C4 ARABIC LETTER WAW WITH RING + U+0622 ARABIC LETTER ALEF WITH MADDA ABOVE]. 

Sources for this section: d, e, u, w.

The set of characters needed to represent the Kashmiri vowels, then, including decompositions and compositions, includes the following. The list may need to be adjusted when the representation of e is better understood.

The Unicode character ٳ [U+0673 ARABIC LETTER ALEF WITH WAVY HAMZA BELOW] represents a valid visual arrangement for Kashmiri, but is deprecated by the Standard, in favour of the decomposed sequence ا + ٟ [U+0627 ARABIC LETTER ALEF + U+065F ARABIC WAVY HAMZA BELOW], because there is no normalisation equivalence.

Vowel absence

Kashmiri uses ْ [U+0652 ARABIC SUKUN] to kill the vowel after a consonant.l2 Note that the shape may resemble an inverted v. This is also called jazm.

Sometimes people use ۡ [U+06E1 ARABIC SMALL HIGH DOTLESS HEAD OF KHAH] for this.

LOC has an example, روْنگ rw°ng, where the vowel absence is indicated over the WAW.

Nasalisation

It seems that ں [U+06BA ARABIC LETTER NOON GHUNNA] may be used for nasalisation of a vowels. To be confirmed. See details about how this works in Urdu.

Palatalisation

Palatalisation is a frequent feature of Kashmiri words. It is represented using ؠ [U+0620 ARABIC LETTER KASHMIRI YEH]. It mostly appears medially and finally: non-final/isolate forms have a small circle beneath them, eg. these are all the same character ؠ  ؠـ  ـؠـ  ـؠ.

Use preceding the inherent vowel seems to typically be transcribed using ê, eg. têmbar. At the end of a word, it is often transcribed using a superscript i, eg. tånⁱ.

Confusables

This section documents some common spelling errors. Most of the illustrations are from a Kashmiri Wikipedia page, but they do occur elsewhere, too.

Kashmiri doesn't use ي [U+064A ARABIC LETTER YEH], but people sometimes use it in Kashmiri text, failing to distinguish it from ی [U+06CC ARABIC LETTER FARSI YEH]. Here is an example from Wikipedia اِنسيكلؤپِيڈيا. The correct spelling would be اِنسیکلۆپِیڈیا.

This can also lead to the use of characters such as ئ [U+0626 ARABIC LETTER YEH WITH HAMZA ABOVE], rather than ی + ٔ [U+06CC ARABIC LETTER FARSI YEH + U+0654 ARABIC HAMZA ABOVE].

That example also uses ك [U+0643 ARABIC LETTER KAF] instead of ک [U+06A9 ARABIC LETTER KEHEH], and ؤ [U+0624 ARABIC LETTER WAW WITH HAMZA ABOVE] instead of ۆ [U+06C6 ARABIC LETTER OE].

Non-final forms of ؠ [U+0620 ARABIC LETTER KASHMIRI YEH] may be incorrectly typed using ٮ + ۪ [U+066E ARABIC LETTER DOTLESS BEH + U+06EA ARABIC EMPTY CENTRE LOW STOP], eg. پٮ۪ٹھ in Wikipedia, rather than پؠٹھ.

(The same page has موسیٖقی, where the final ی [U+06CC ARABIC LETTER FARSI YEH] should perhaps be ؠ [U+0620 ARABIC LETTER KASHMIRI YEH] ?)

Consonant clusters & gemination

The diacritic ّ  [U+0651 ARABIC SHADDA] doubles the value of the consonant it is attached to. It is also used to explicitly signal an absence of intervening vowels between any two consonants, where needed.

Formatting characters

The Arabic script uses a large number of Unicode characters that affect the way that other characters are rendered. Many of those have no visible form of their own.

Modern Arabic-script text makes use of a relatively large set of invisible formatting characters, especially in plain text, many of which are used to manage text direction. For more details, see the Arabic overview.
