Usage & history

For information about the script in general, see the Arabic overview.

Hausa can be written in the Latin script, but also (less commonly) using the Arabic ajami script.

See the comparison table

Distinctive features

The Arabic script is an abjad. This means that in normal use the script represents only consonant and long vowel sounds. The Arabic alphabet has been adapted to cope with the many more vowels sounds in languages such as Hausa. See the table to the right for a brief overview of features, taken from the Script Comparison Table.

Hausa is principally written using the naskh style of Arabic writing. It includes several characters and diacritics that are not used in standard Arabic.

Unlike Standard Arabic, ajami tends to add all vowel diacritics to text.

Structure

Hausa has 3 syllable types: CV, CVV, and CVC, where VV can be a long vowel or a diphthong.c The long vs. short vowel distinction is phonemically important, however when a syllable with a long vowel acquires and final consonant, the vowel is shortened.

Consonant clusters may occur where syllables are side by side, but not within a syllable. Gemination is, however, a distinctive feature.c

Semivowels ʷ and ʲ may occur after an initial consonant.

Vowel absence

Hausa uses ْ [U+0652 ARABIC SUKUN] to kill the vowel after a consonant, eg. تَرْسَشِنْ tar͓saʃin͓.

Vowel absence is usually marked (unlike Standard Arabic), including over the YEH or WAW that signal the final part of a diphthong.

Consonants

Phoneme table

There is no absolute standard for how to write African languages in ajami, so there is some variation in which characters are mapped to which sounds.

There are two main styles of writing, Ḥafṣ and Warš, which involve the use of different characters for certain phonemes. In the table, Ḥafṣ-style characters are preceded by h, and Warš by w.

In addition, alternative shapes, sometimes those used for other sounds, may be used in borrowed words, or because sounds are not differentiated in some regions.e These characters are preceded by an asterisk in the table.

Warš orthography

The list below shows variants used for the Warš orthography.

ࢼ [U+08BC ARABIC LETTER AFRICAN QAF] differs from ق [U+0642 ARABIC LETTER QAF] in two respects: it has no dot for the isolate and final positions, and when it does have a dot it appears below the base, eg. ق ققق vs. ࢼ ࢼࢼࢼ.

ࢻ [U+08BB ARABIC LETTER AFRICAN FEH] differs from ف [U+0641 ARABIC LETTER FEH] in the same way, eg. ف ففف vs. ࢻ ࢻࢻࢻ.

The same applies to ࢽ [U+08BD ARABIC LETTER AFRICAN NOON] and ن [U+0646 ARABIC LETTER NOON], eg. ن ننن vs. ࢽ ࢽࢽࢽ.

ك [U+0643 ARABIC LETTER KAF] and ک [U+06A9 ARABIC LETTER KEHEH] look the same in the webfont used for this page, but represent different underlying characters.

The other two characters have a triple-dot addition which is associated with glottalised consonants in the Warš orthography. (They don't appear to have glyphs in the webfont used.)

Consonant clusters & gemination

Geminated consonants are indicated using ّ [U+0651 ARABIC SHADDA], eg. بَبَّنْ baba˖n͓ big.

Modern Arabic-script text makes use of a relatively large set of invisible formatting characters, especially in plain text, many of which are used to manage text direction. For more details, see the Arabic overview.

Orthographic development & variants

Hausa has also been written in ajami since the early 17th century.w

There is no standard system of using ajami for Hausa, and different writers may use letters with different values.w

There are or have been a number of variant practices for writing Hausa ajami. There are also some confusable characters. They include the following:

  • There is some variation in sources about the use of ی [U+06CC ARABIC LETTER FARSI YEH] versus ي [U+064A ARABIC LETTER YEH]. Most sources seem to agree, however, that the long i uses ى [U+0649 ARABIC LETTER ALEF MAKSURA], whereas the end of a diphthong uses one of the former.
