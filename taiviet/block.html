This page lists characters in the following Unicode block and provides information about them.

This is not authoritative, peer-reviewed information – these are just notes I have gathered and copied from various places.

For details about how the script works, see Tai Viet/Tai Dam. For further information related to this and other scripts, see the Script links pages.

Read more about how to use this page.

To find a character by codepoint, type #char0000 at the end of the URL in the address bar, where 0000 is a four-figure, hex codepoint number, all in uppercase. Or type the character or the hex number in the Find control above.

To view this page as intended, you need a Tai Viet font. This page comes with a Tai Viet webfont. Click the blue vertical bar at the bottom right of the page to apply other fonts, if you have them on your system. For transcriptions I recommend the excellent and free Doulos SIL font. The large character in the box will not be rendered unless the webfont downloaded with the page or a system font has a glyph for it. (If there is no glyph and you want to see what it looks like, click on See in UniView.)

U+AAA5 TAI VIET LETTER HIGH YO

See in UniView

Tai Dam j

This letter is the one used for syllable final position.

The traditional sequence ꪵ-ꪫ is ambiguous in open syllables. It can be interpreted as either Cɛw, or as Cʷɛ (where C represents a velar consonant). To eliminate this ambiguity, the character ꪥ [U+AAA5 TAI VIET LETTER HIGH YO] is sometimes appended to the end of the sequence to indicate the second pronunciation eg. ꪵꪁꪫꪥ ɛkwy kʷɛ² cinnamon. Since j never occurs after ɛ, this can be done without creating a new ambiguity. This spelling is only used in some dialects of the traditional script. However, it has been adopted as a standard spelling in a project sponsored by the Son La Department of Science and Technology. b6

General sources: Unicode, Brase

See in UniView

Tai Dam v, -w

This letter is the one used for syllable final w, eg. ꪔꪲꪫ ṯiw tʰiw¹ to whistle.

This letter is also used to labialise initial velar consonants when appearing immediately after the consonant. b4 The placement of a vowel sign can help avoid ambiguity here: compare ꪀꪲꪫ ḵiw kiw vs. ꪀꪫꪲ ḵʷi kʷi. b5

Also occurs in the following vowel:

ʷɛ ꪵ-ꪫ, where the missing consonant is velar, and the result is a labialised velar consonant followed by ɛ, eg. ꪵꪁꪫꪥ ɛkwy kʷɛ² cinnamon.

The traditional sequence ꪵ-ꪫ is ambiguous in open syllables. It can be interpreted as either Cɛw, or as Cʷɛ (where C represents a velar consonant). To eliminate this ambiguity, the character ꪥ [U+AAA5 TAI VIET LETTER HIGH YO] is sometimes appended to the end of the sequence to indicate the second pronunciation (as in the earlier example). Since j never occurs after ɛ, this can be done without creating a new ambiguity. This spelling is only used in some dialects of the traditional script. However, it has been adopted as a standard spelling in a project sponsored by the Son La Department of Science and Technology. b6

Alternatively, the position of a tone mark can help disambiguate the pronunciation. Compare ꪵꪀ꫁ꪫ ɛḵ²w kɛw³ and ꪵꪀꪫ꫁ ɛḵw² kʷɛ³. b5

Tone marks

Combining tone marks

꪿

U+AABF TAI VIET TONE MAI EK

See in UniView

Tai Dam mai ek

Used only on unchecked syllables (ie. not ending in -p, -t, or -k).

Sets tone 5 for high onsets, eg. TBD, and tone 2 for low, eg. ꪄꪴ꪿ꪙ x̱u¹n xun² dust.

Combining tone marks were originated by Tai Dam speakers in the United States, using Lao tone marks. The Tai community in Vietnam, however, invented their own spacing tone marks written on the base line at the end of the syllable. This mark corresponds to ꫀ [U+AAC0 TAI VIET TONE MAI NUENG]. b7

General sources: Unicode, Brase
